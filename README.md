# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is currently not compatible with SWC. See [this issue](https://github.com/vitejs/vite-plugin-react/issues/428) for tracking the progress.

# 🔍 Pokédex Interactiva - React & PokéAPI

Este proyecto es una Pokédex moderna y responsiva que consume datos de la [PokéAPI](https://pokeapi.co/). Permite explorar, buscar y filtrar Pokémon de manera fluida.

## ✨ Características
* **Diseño Adaptativo**: Visualización optimizada para escritorio (arte oficial) y móvil (GIFs animados).
* **Búsqueda Dinámica**: Barra de búsqueda con estilizado personalizado en color amarillo Pokémon.
* **Filtrado por Tipo**: Menú desplegable para filtrar la lista según el tipo elemental.
* **Interfaz Limpia**: Tarjetas compactas con nombres, IDs y etiquetas de tipo alineadas.

## 🛠️ Tecnologías
* **React**: Biblioteca principal para la interfaz.
* **Material UI (MUI)**: Sistema de diseño y componentes.
* **Axios**: Cliente HTTP para las peticiones a la API.
